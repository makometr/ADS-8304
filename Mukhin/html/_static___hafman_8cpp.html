Функция декодирования закодированной строки.

Декодирование происходит до тех пор, пока мы не пройдёмся по закодированной строке полностью. В цикле перебирая все пары из словаря result мы ищем ключ-символ, строка-значение которого будет началом закодированной строки. В виду того, что все коды символов обладают свойством префиксности, проход по закодированной строке будет однозначным.

Parameters
resultсловарь <char, std::string>, ключ является символом, а значение по ключу кодированием этого символа
coding_messageзакодированное сообщение
iпорядковый номер аргумента командной строки, нужен для создания очередного файлы вывода
