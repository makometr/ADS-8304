Decoding algorithm

For decoding, you can use the constructed tree or get character codes from a file. Decoding Algorithm:

  1. It is created using a tree, or a dictionary of the form "symbol_code, symbol" is read from a file
    2. In the loop:
  2. The next bit is read, placed in the line-buffer.
  3. Go to the next bit.
  4. If there is an existing character code in the buffer line: the symbol from the dictionary with the given code is read and written to the de-encoding result, the buffer line is cleared.
